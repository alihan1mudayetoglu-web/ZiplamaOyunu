# Zıplama Oyunu

Karakterin zıplayarak engelleri aşmasını sağlayan oyundur.

## Açıklama

Yer çekimi ve zıplama mekaniklerini içeren bir platformer oyun. Karakterin engelleri aşması, platfromlara konması ve belirli seviyelerden geçmesi gerekir. Klasik platform oyun temellerini içerir.

## Kullanım

1. Python 3'ü kur
2. Gerekli kütüphaneyi yükle: `pip install pygame`
3. `ZiplamaOyunu.py` dosyasını çalıştır
4. Oyuna başla

## Kontroller

- **Sol/Sağ Ok**: Sola/Sağa hareket
- **SPACE veya Yukarı Ok**: Zıpla
- **ESC**: Oyundan çık
